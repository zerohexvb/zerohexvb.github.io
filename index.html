Mallox Ransomware: Tactics, Evolution & Technical Analysis

Mallox (aka TargetCompany) ransomware is targeting unsecured Microsoft SQL Servers to gain access and deploy its payload. Multiple failed login attempts indicate MSSQL brute force attacks as the primary entry method. Once access is obtained, it uses ‘sqlservr.exe’ to execute and distribute malicious files on the victim’s system.

Malware Campaign Leveraging SVGs & BAT Scripts to Drop XWorm & Remcos

Recent campaigns use obfuscated BAT-based loaders delivered via ZIP archives hosted on trusted-looking platforms to deploy RATs like XWorm and Remcos. These BAT scripts trigger PowerShell loaders that inject payloads directly into memory for fileless execution. Attackers are also abusing SVG files with embedded JavaScript, marking a shift toward non-traditional formats for malware delivery.

Spear Phishing Campaign Delivers VIP Keylogger

A new campaign is distributing the VIP keylogger through spear-phishing, but this time using an AutoIt-based injector instead of previous delivery methods. Once executed, the injector deploys the VIP keylogger, which steals credentials from major browsers and logs keystrokes. Attackers favor AutoIt for its easy obfuscation and ability to compile scripts into AV-evading executables.

DarkRace Ransomware: Deep Dive into its Techniques & Impact

DarkRace ransomware, inspired by leaked Lockbit code, encrypts files using Salsa20, deletes shadow copies, and terminates processes to maximize damage. It uses mutexes, XML-based configurations, and whitelisting to avoid reinfection and skip critical files. Post-encryption, it removes traces by deleting event logs, dropped files, and restarting the system, highlighting the need for vigilant cybersecurity measures.
